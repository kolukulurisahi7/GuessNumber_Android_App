# GuessNumber Android App

A simple Android mobile application built with Android Studio using Kotlin and XML.

## Description

GuessNumber is a basic game where the app generates a random number between 1 and 100. The user enters a number and taps **Guess**. The app then shows whether the guess is too low, too high, or correct.

## Features

- Random number between 1 and 100
- Text field for user input
- Guess button
- Result message
- New Game button to restart

## Tools Used

- Android Studio
- Kotlin
- XML
- Android Emulator

## How to Run

1. Open the project in Android Studio
2. Start an emulator or connect an Android device
3. Click the Run button
4. Enter a number and test the app

## Files

- `MainActivity.kt` - main app logic
- `activity_main.xml` - user interface layout
